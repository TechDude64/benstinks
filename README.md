# Ben Chadwick: A Legend in Stink 🏆

A humorous personal website celebrating the legendary aromatic achievements of Ben Chadwick.

## Features

- **Biography** - The origin story of the stink legend
- **Greatest Stinks** - Hall of fame for his most potent moments
- **Daily Odor Log** - Real-time tracking of his aromatic levels
- **Testimonials** - What people really think (ouch)
- **Photo Gallery** - Evidence of his legendary status

## Tech Stack

- Pure HTML5
- CSS3 with custom properties and animations
- No frameworks required

## How to View

Simply open `index.html` in any web browser. No build process or server needed!
