# Ben Chadwick: A Legend in Stink 🏆

A humorous React website celebrating the legendary aromatic achievements of Ben Chadwick.

## Features

- **Biography** - The origin story of the stink legend
- **Greatest Stinks** - Hall of fame for his most potent moments
- **Daily Odor Log** - Real-time tracking of his aromatic levels
- **Testimonials** - What people really think (ouch)
- **Photo Gallery** - Evidence of his legendary status

## Tech Stack

- React 19 + TypeScript
- Vite for fast development
- CSS with custom properties and animations

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
